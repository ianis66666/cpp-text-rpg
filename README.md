# C++ Text RPG

這是 C++ 文字版 RPG 小遊戲作業倉庫。

## 作業需求

- 使用 C++ 製作 CLI 文字遊戲
- 至少包含四個 class：
  - `Player`
  - `Skill`
  - `Item`
  - `Monster`
- 遊戲需要可以編譯、執行、遊玩
- Word 文件需要說明玩法、class 設計、分工與執行截圖

## 建議分工

- Player class：角色名稱、血量、攻擊力、道具、攻擊與受傷判斷
- Skill class：技能名稱、傷害、MP 消耗、技能資訊
- Item class：物品名稱、類型、效果、數量、使用判斷
- Monster class：怪物名稱、血量、攻擊力、金幣獎勵、戰鬥流程

## 基本操作設計

遊戲可設計成回合制戰鬥：

1. 普通攻擊
2. 使用技能
3. 使用物品
4. 查看狀態

玩家打敗怪物即獲勝，玩家 HP 歸零則失敗。
